# RAG
This repository contains all the notebooks related to my undergraduate thesis.
There are 3 notebooks. 
1) RAFT Dataset Preparation : It contains the notebook used to create custom dataset to train LLM for RAFT. RAFT is a technique where LLM is trained for information retrieval.
2) LLM Fine-Tuning : This Notebook is used to Fine-tune LLM ( Specifically Gemma-2-9b-it).
3) RAG Pipeline & Evaluation : This is the main notebook which contains RAG pipeline & also RAGas evaluation methodology.
Besides the 3 Notebooks there are two files.
1) RAFT Q/A dataset.
2) For LLM Fine-tuned weights and biases please go to : https://drive.google.com/file/d/1qQeBJ1QA3rL0tfKDPQ18dUNrWQoewP0E/view?usp=sharing
